Some thoughts

When I built this website I used mobile-first approach. I hand coded everyhing myself, one reason is to be able to actually include my portfolio in my...portfolio. I'm not really against using Bootstrap for example, I just prefer to do it all myself. When using the navigation I wanted the scroll to be smooth. There is a css property called scroll-behaviour which would have solved the problem BUT the browser support at the time is not good at all, strangely enough. I found some jQuery solutions but instead I wrote my own vanilla javascript to make it work, full control! I am also very pleased with the not so symmetrical X when the menu is shown.
